# Contributing

Thanks for helping improve Place Me in the World Cup.

## Local Development

This is a static site. You can open `index.html` directly or run:

```bash
python3 scripts/serve.py
```

Then open `http://127.0.0.1:4321/`.

## Project Shape

- Edit prompt templates in `data/scenarios.js`.
- Edit team jersey descriptions in `data/teams.js`.
- Put card thumbnails and preview clips in `assets/thumbs/`.
- Keep generated research/source files out of commits unless they are needed by the site.

## Pull Requests

Please keep changes focused and include a quick note about how you tested the app.
